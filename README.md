# gehirn
Mein kleines Wiki. Für all den Krempel der so täglich auf mich einströmt, und den ich möglicherweise irgendwann wieder brauche.

## Installation

Nicht notwendig. Eine Möglichkeit das Wiki im Browser zu betrachten wäre [madness](https://github.com/DannyBen/madness), ein Ruby-Gem welches alle Markdown-Dateien im Arbeitsverzeichnis (rekursiv) auf `localhost:3000` liefert.

``` bash
gem install madness
madness
```
